仕事内容

JOB DESCRIPTION

ホールスタッフ

お客様のご案内、オーダー、お料理の提供、レジ対応など、接客全般をお任せします。 まずは笑顔で「いらっしゃいませ」が言えればOK！

  • 未経験でも安心の研修あり
  • 接客スキルが身につく

キッチンスタッフ

簡単な仕込みや盛り付け、調理補助、食器洗いなどをお任せします。 マニュアルがあるので、料理未経験でも安心してスタートできます。

  • 料理の基礎が学べる
  • 接客が苦手でも活躍できる

パティシエ・製造

ケーキや焼き菓子の製造、新商品の開発補助を行います。 お菓子作りが好きな方なら、実務経験がなくてもイチから教えます。

  • お菓子作りの技術習得
  • 商品開発に携われる
